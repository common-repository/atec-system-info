atec System Info (Operating system, server, memory, PHP and database details)

== Description ==

This plugin provides detailed system information, such as operating system, server, memory, PHP and database details. It will also show PHPinfo, php.ini and PHP extensions.

Lightweight (63KB) and resource-efficient.
Backend CPU footprint: <1 ms.
Frontend CPU footprint: <1 ms.

Terms: https://ipinfo.io/terms-of-service

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or through the `Plugins` menu.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Click "atec System Info" link in admin menu bar.
